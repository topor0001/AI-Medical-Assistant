# Коррекция доз: [Препарат / группа препаратов]

> **Дисклеймер**: Рекомендации основаны на инструкциях и КР. Требуется индивидуальный подход.

## Почечная недостаточность
| СКФ (мл/мин/1.73 м²) | Рекомендация по дозе |
|----------------------|----------------------|
| > 60 | стандартная доза |
| 30–60 | снижение на 50% |
| 15–30 | 25% от стандартной |
| < 15 | противопоказан / применять с осторожностью |

## Печёночная недостаточность (класс Child-Pugh)
| Класс | Рекомендация |
|-------|---------------|
| A (лёгкая) | без коррекции |
| B (средняя) | снижение дозы на 50% |
| C (тяжёлая) | противопоказан / применять с осторожностью |

## Пожилой возраст (≥ 65 лет)
- Начинать с минимальной дозы
- Титровать медленно
- Контролировать функцию почек
