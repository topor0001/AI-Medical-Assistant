# MedAssist

## База знаний интеллектуального медицинского ассистента

Проект **MedAssist** представляет собой базу знаний интеллектуальной медицинской системы, предназначенной для структурированного хранения, систематизации, связывания и последующего использования медицинской информации.

Тема курсового проекта: **«База знаний интеллектуального медицинского ассистента»**.

Разрабатываемая база знаний рассматривается как информационно-справочное ядро интеллектуальной системы, поддерживающей диагностический поиск, представление медицинских сведений, навигацию по связанным сущностям и последующую интеграцию с прикладными модулями.

## Цель проекта

Цель проекта — разработка базы знаний интеллектуального медицинского ассистента MedAssist, обеспечивающей:

- структурированное хранение медицинских знаний;
- логическое связывание симптомов, заболеваний, обследований и опасных состояний;
- поддержку диагностического поиска;
- основу для последующей реализации интеллектуальной медицинской системы.

## Предметные области системы

В архитектуре MedAssist выделяются три специализированные предметные области:

1. **Заболевания, симптомы и диагностика**
2. **Фармакология и терапия**
3. **Клинические протоколы и маршрутизация пациента**

В рамках текущей части проекта основное внимание уделяется предметной области:

**«Заболевания, симптомы и диагностика»**

Данный контур включает:
- симптомы и синдромы;
- заболевания;
- методы обследования;
- дифференциальную диагностику;
- красные флаги;
- неотложные состояния.

## Формат представления знаний

База знаний проектируется в документно-ориентированном формате.

Каждая единица знаний хранится как отдельный **Markdown-документ** с обязательным блоком **YAML-метаданных**. Такой подход обеспечивает:

- человекочитаемость;
- единообразие структуры;
- удобство сопровождения;
- пригодность к автоматизированной обработке;
- возможность построения явных связей между документами.
